## STORY

히로인들은 비록 은 나쁘게 자신의 차원에서 떨어졌지만, 다시 원래 세계에 돌아가기 위해 콜로세움에서 자신의 위용을 뽐내며 1등을 해서 메테오 결정을 만들어 만다. 콜로세움에 상기하자마자 시작된 것은 무제한 격투. 우선 이기고 보자.

## 멀티플레이

### MIRROR 네트워크

Mirror 네트워크를 통해 플레이어 매칭 및 네트워크를 정확히 동기화한다.
각 클라이언트의 입력과 변수, 게임 시간 등을 일관성 있게 반영한다.

### 핵심 특징

- **Easy to Play** — 복잡한 커맨드 없이 키 하나로 매핑된 직관적인 전투
- **빠른 템포** — 5분 내외의 짧은 매치로 자투리 시간에 즐길 수 있는 게임
- **전략적 깊이** — 아이템과 지형지물을 활용한 다채로운 전략 구사

---

## 맵 소개

3D 다층 구조의 넓은 맵을 통해 여러가지 전투방식을 구사할 수 있다.
상대방과 거리를 벌려 원거리 데미지를 구사하거나, 맵 오브젝트를 활용해 높은 위치를 선점할 수 있다.

### 지형지물

| 지형지물 | 효과 |
|---|---|
| **점프대** | 밟으면 위로 높이 솟아오름 |
| **폭포** | 거슬러 올라가거나 천천히 낙하 |
| **장외 구역** | 닿으면 체력 50% 피해 |

## 플레이어

플레이어는 이동 및 점프, 회피를 통해 움직일 수 있다.
기본 공격 및 강공격을 이용해 상대를 공격할 수 있다.

### 조작법

| 동작 | 키 입력 |
|---|---|
| 이동 | `W` `A` `S` `D` `Space` |
| 대시 | `Shift` |
| 기본 공격 (콤보) | 마우스 좌클릭 |
| 강공격 (차지) | 마우스 우클릭 길게 |
| 앉기 | `C` |
| 아이템 획득 | `E` |
| 필살기 사용 | `R` |
| 액티브 아이템 | `Q` |
| 근접 무기 던지기 | 마우스 우클릭 |
| 뒤로가기 | `ESC` |

## 아이템 소개

맵 곳곳에 배치된 다양한 아이템을 활용한 짜릿한 역전을 노려보자.

### 아이템 카테고리

| 카테고리 | 아이템 | 설명 |
|---|---|---|
| **무기 아이템** | 활, 점착 폭탄, 단검, 테이저 건, 도끼, 검, 방망이 | 각 무기마다 사거리·피해량·특수 효과가 다름 |
| **액티브 아이템** | 덫, 스팀팩, 연막탄 | 원하는 타이밍에 발동시켜 변수를 만듦 |
| **필드 아이템** | 미사일, 중력변환장치, 독안개 | 맵 전체에 영향을 주는 강력한 효과 |
| **패시브 아이템** | 질주, 배터리, 달팽이 | 획득 즉시 캐릭터 능력치를 변화 |

### 아이템 플레이의 장점

- **실력 격차 완화** — 강력한 아이템으로 역전의 기회를 제공
- **변수 창출** — 매 경기마다 다른 양상의 전투
- **시너지의 즐거움** — 아이템 조합으로 전략적 승리
