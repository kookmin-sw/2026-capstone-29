## 게임 소개

> *3D 캐주얼 격투 게임*

### 핵심 특징

- **Easy to Play** — 복잡한 커맨드 없이 키 하나로 매핑된 직관적인 전투
- **빠른 템포** — 5분 내외의 짧은 매치로 자투리 시간에 즐길 수 있는 게임
- **전략적 깊이** — 아이템과 지형지물을 활용한 다채로운 전략 구사

## 아이템 소개

맵 곳곳에 배치된 다양한 아이템을 활용한 짜릿한 역전을 노려보자.
### 아이템 카테고리

#### 무기 아이템
활, 점착 폭탄, 단검, 테이저 건, 도끼, 검, 방망이 — 각 무기마다 사거리·피해량·특수 효과가 다릅니다.

#### 액티브 아이템
덫, 스팀팩, 연막탄 — 원하는 타이밍에 발동시켜 변수를 만듭니다.

#### 필드 아이템
미사일, 중력변환장치, 독안개 — 맵 전체에 영향을 주는 강력한 효과를 가집니다.

#### 패시브 아이템
질주, 배터리, 달팽이 — 획득 즉시 효과가 적용되어 캐릭터의 능력치를 변화시킵니다.

> 🖼️ *아이템 활용 플레이 장면 이미지 추가 예정*

### 아이템 플레이의 장점

- **실력 격차 완화** — 강력한 아이템으로 역전의 기회를 제공
- **변수 창출** — 매 경기마다 다른 양상의 전투
- **시너지의 즐거움** — 아이템 조합으로 전략적 승리

### 지형지물

| 지형지물 | 효과 |
|---|---|
| **점프대** | 밟으면 위로 높이 솟아오름 — 빠른 수직 이동 가능 |
| **폭포** | 거슬러 올라가거나 천천히 낙하 — 회피·기동 활용 |
| **장외 구역** | 닿으면 체력 50% 피해 — 낙사 유도 플레이 가능 |
